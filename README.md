# 강의 슬라이드 → 강의 원고 변환기 (slides2manuscript)

교수님 **슬라이드 PDF**를 넣으면, 발표 대본이 아니라 **사람이 직접 쓴 듯한 줄글 강의 원고(.docx)**
로 바꿔 주는 프로그램입니다.

> 강사분들의 발표 슬라이드를 바탕으로 강의 원고를 제출해야 하는 조교분들의 수고를 덜기
> 위해 만들었습니다. 문의는 **기술분과 조교 백승렬**(010-9265-8369, paik402@snu.ac.kr)으로
> 주시면 됩니다.

- 입력: **슬라이드는 반드시 PDF로 넣어야 합니다** (텍스트 기반 PDF, 파워포인트·키노트에서 내보낸 것)
- 출력: A4 **25~30쪽** 분량의 워드 문서(.docx), 꾸밈 없는 깔끔한 본문
- 특징: 슬라이드 내용에 충실 · 개조식(글머리표) 없이 줄글 · AI 티 안 나는 문장

---

## 설치·사용 순서

처음 한 번만 설치와 키 발급이 필요하고(약 10~15분), 그다음부터는 명령어 한 줄이면 됩니다.

1. 파이썬(Python) 설치
2. 이 프로그램 내려받기
3. 라이브러리 설치
4. API 키 발급
5. 키를 `.env` 파일에 넣기
6. 실행

> API 키는 AI(클로드 또는 GPT)를 호출할 때 쓰는 인증 키입니다.
> 본인이 발급받아야 하며, 사용한 만큼 요금이 부과됩니다.

---

## 1단계. 파이썬 설치

### 맥(Mac)
1. https://www.python.org/downloads/ 접속 → 노란 **Download Python** 버튼 클릭.
2. 내려받은 `.pkg` 파일을 더블클릭 → "계속 / 동의 / 설치" 를 끝까지 누름.

### 윈도우(Windows)
1. https://www.python.org/downloads/ 접속 → **Download Python** 버튼 클릭.
2. 내려받은 설치 파일 실행.
3. ⚠️ **중요**: 첫 화면 맨 아래 **"Add Python to PATH"** 체크박스를 반드시 켠 뒤
   **Install Now** 클릭.

설치 확인(선택): 터미널(맥) 또는 명령 프롬프트(윈도우)를 열고 아래를 입력해
버전 숫자가 나오면 성공입니다.
```
python3 --version
```
(윈도우에서 안 되면 `python --version` 으로 시도)

> **터미널/명령 프롬프트 여는 법**
> - 맥: `Command(⌘) + 스페이스` → "터미널" 입력 → 엔터
> - 윈도우: 시작 메뉴에서 "cmd" 또는 "PowerShell" 검색 → 실행

## 2단계. 이 프로그램 내려받기

### 방법 A — ZIP로 받기 (제일 쉬움, 추천)
1. 이 페이지 위쪽의 초록색 **`< > Code`** 버튼 클릭.
2. **Download ZIP** 클릭.
3. 내려받은 압축 파일을 풀기(더블클릭). 예: 바탕화면에 `snu_ai_doc` 폴더가 생김.

### 방법 B — git 으로 받기 (git을 아는 경우)
```
git clone https://github.com/paiiek/snu_ai_doc.git
```
> 나중에 프로그램이 업데이트되면 이 폴더에서 `git pull` 한 줄로 최신본을 받습니다.

---

## 3단계. 필요한 부품(라이브러리) 설치

1. 터미널/명령 프롬프트를 엽니다.
2. 프로그램 폴더로 이동합니다. **`cd ` (씨디+한 칸 띄우기) 까지만 친 뒤,
   폴더를 터미널 창으로 드래그**하면 경로가 자동으로 입력됩니다. 그리고 엔터.
   ```
   cd /여기에/드래그된/snu_ai_doc/경로
   ```
3. 아래 한 줄을 입력하고 엔터(설치에 1~2분):
   ```
   pip3 install -r requirements.txt
   ```
   (윈도우에서 `pip3` 가 안 되면 `pip install -r requirements.txt`)

---

## 4단계. API 키 발급받기

**클로드(Anthropic)와 GPT(OpenAI) 중 하나만** 있으면 됩니다. 가진 계정을 쓰세요.

### 클로드(Anthropic) 키 — 기본 권장
1. https://console.anthropic.com 접속 → 가입/로그인.
2. 왼쪽 메뉴 **Billing** 에서 결제수단 등록(선불 충전식. 소액 충전 가능).
3. 왼쪽 메뉴 **API Keys** → **Create Key** → 이름 아무거나 → 생성.
4. 나온 키(`sk-ant-...`)를 복사해 둡니다. **이 화면을 닫으면 다시 못 보니 꼭 복사.**

### GPT(OpenAI) 키 — 클로드 안 쓰는 경우
1. https://platform.openai.com/api-keys 접속 → 로그인.
2. 결제수단 등록(Billing) 후 **Create new secret key** → 키(`sk-...`) 복사.

> 💰 **비용 안내**: 강의 한 편(슬라이드 수십 장 → 약 27쪽 원고) 변환에 모델 호출
> 요금이 듭니다. 정확한 금액은 모델·분량에 따라 다릅니다. 비용을 아끼려면
> 6단계에서 `--model claude-sonnet-4-6`(클로드) 같은 저렴한 모델 옵션을 쓰세요.

---

## 5단계. 키를 `.env` 파일에 넣기

1. 프로그램 폴더 안의 **`.env.example`** 파일을 복사해서 이름을 **`.env`** 로 바꿉니다.
   (점으로 시작하는 파일명 그대로. 확장자 없음.)
2. `.env` 파일을 메모장(윈도우)/텍스트편집기(맥)로 엽니다.
3. 본인 키에 맞는 줄의 값만 바꿔 저장합니다.
   - 클로드를 쓸 경우:
     ```
     ANTHROPIC_API_KEY=sk-ant-여기에-본인-키
     ```
   - GPT를 쓸 경우: 위 줄은 `#` 로 막고, OpenAI 줄의 `#` 를 지운 뒤 키를 넣습니다.
     ```
     OPENAI_API_KEY=sk-여기에-본인-키
     ```

> 🔒 `.env` 파일은 **본인 컴퓨터에만** 있고 GitHub에 올라가지 않습니다(자동 차단됨).
> 키는 비밀번호이니 누구에게도 보내지 마세요.

---

## 6단계. 실행하기

1. 변환할 **슬라이드 PDF를 프로그램 폴더 안에 복사**해 넣습니다.
2. 터미널에서(3단계처럼 폴더로 이동한 상태에서) 아래를 입력합니다.
   따옴표 안에 **본인 PDF 파일 이름**을 넣으세요:
   ```
   python3 make_manuscript.py "강의1.pdf"
   ```
   (윈도우에서 `python3` 가 안 되면 `python make_manuscript.py "강의1.pdf"`)
3. 진행 상황이 화면에 표시되고(몇 분 소요), 끝나면 같은 폴더에
   **`강의1_원고.docx`** 가 생깁니다. 워드/한글로 열어 확인하세요.

### 자주 쓰는 옵션
```
python3 make_manuscript.py "강의1.pdf" -t "1주차: 인공지능 개요" -o "1주차_원고.docx"
```
| 옵션 | 뜻 | 예시 |
|---|---|---|
| `-t` | 원고 맨 위 제목 | `-t "1주차 강의"` |
| `-o` | 저장할 파일 이름 | `-o "1주차_원고.docx"` |
| `--pages` | 목표 분량(쪽). 기본 27 | `--pages 28` |
| `--model` | 사용할 AI 모델(비용 절감용) | `--model claude-sonnet-4-6` |
| `--provider` | `anthropic` 또는 `openai` 강제 지정 | `--provider openai` |

전체 옵션 보기: `python3 make_manuscript.py --help`

> 분량은 **25~30쪽 사이에 들어오도록** 프로그램이 자동으로 맞춥니다(부족하면 늘리고
> 넘치면 줄임). 슬라이드 내용 자체가 너무 적으면 25쪽에 못 미칠 수 있습니다.

## 문제가 생기면 (자주 나오는 오류)

| 화면에 나오는 말 | 뜻과 해결 |
|---|---|
| `command not found: python3` | 파이썬이 안 깔렸거나 PATH 미설정. 1단계 다시(윈도우는 "Add to PATH" 체크). |
| `No module named ...` | 3단계(`pip install -r requirements.txt`)를 안 했거나 다른 폴더에서 실행. 폴더 이동 후 다시. |
| `API 키가 없습니다` | 5단계 `.env` 설정 확인. 파일명이 정확히 `.env` 인지(`.env.txt` 아님) 확인. |
| `이미지/스캔 PDF로 의심됩니다` | 텍스트가 안 들어 있는 PDF입니다. 이 도구는 텍스트 기반 PDF 전용입니다. |
| `입력 파일을 찾을 수 없습니다` | PDF가 폴더 안에 있는지, 따옴표 안 이름이 정확한지 확인. |

## 꼭 기억할 점

- 생성된 원고는 **초안**입니다. **반드시 사람이 한 번 읽고 검토**한 뒤 사용하세요.
  슬라이드에 있는 내용만 근거로 쓰므로, 슬라이드가 부실하면 원고도 부실합니다.
- 수식·다이어그램·표·그림은 텍스트로 바뀌는 범위에서만 반영됩니다.
- API 키와 슬라이드 PDF는 외부에 공유하지 마세요.
